# Orçamento Familiar Detalhado

Este é um sistema simples para gerenciar orçamentos familiares, incluindo planejamento, registro de despesas e resumo com gráfico.

## Como Usar

1. Abra o arquivo `index.html` no navegador.
2. Adicione categorias e valores planejados.
3. Registre despesas diárias.
4. Visualize o resumo e o gráfico comparativo.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Chart.js (para gráficos)
